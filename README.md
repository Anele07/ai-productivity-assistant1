# TaskPilot AI Suite

> **Your workday, on autopilot.**

TaskPilot AI Suite is an AI-powered workplace productivity platform that helps professionals automate repetitive office tasks through a single, intelligent workspace. Instead of using multiple applications, users can generate professional emails, summarize meeting notes, plan tasks, conduct research, and interact with an AI workplace assistant—all within one modern dashboard.

This project was developed as part of the **CAPACITI AI Skill Accelerator Programme** to demonstrate practical AI implementation, prompt engineering, modern UI/UX design, and responsible AI usage.

---

# Project Overview

Modern workplaces involve repetitive tasks such as writing emails, summarizing meetings, organizing schedules, researching information, and managing daily work. These activities consume valuable time and often require switching between multiple applications.

TaskPilot addresses this challenge by combining several AI-powered productivity tools into one integrated platform. The application provides an intuitive dashboard where users can complete workplace tasks more efficiently using Generative AI.

# Setup Instructions

## 1. Clone the repository

```bash
git clone https://github.com/your-username/TaskPilot.git
```

## 2. Navigate to the project folder

```bash
cd TaskPilot
```

## 3. Install project dependencies

```bash
npm install
```

## 4. Configure environment variables

Create a `.env` file and add your API keys.

Example:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
VITE_GEMINI_API_KEY=your_gemini_api_key
```

## 5. Start the development server

```bash
npm run dev
```

## 6. Open the application

Open your browser and visit:

```
http://localhost:5173
```
